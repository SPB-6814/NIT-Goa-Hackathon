## For Users Creating Content

### Creating a Tagged Post

1. **Open Create Post Dialog**
   - Click "Create Post" button in the left sidebar
   - Dialog opens

2. **Write Your Content**
   - Type your message in the text area
   - Add images if desired (optional, max 4)

3. **Select Tags** â­ NEW!
   - Scroll down to the "Tags" section
   - You'll see 6 colorful badge buttons:
     - Technical
     - Cultural
     - Academic
     - Social
     - Sports
     - Other
   
4. **Click One or More Tags**
   - Click any tag to select it (it will highlight)
   - Click again to deselect
   - You can select multiple tags
   - **At least one tag is required**

5. **Post!**
   - Click the "Post" button
   - Your post will be created with the selected tags

**Example:**

### Creating a Tagged Project

1. **Open Create Project Dialog**
   - Click "Create Project" button in the left sidebar

2. **Fill Project Details**
   - Enter project title
   - Write description
   - Add required skills

3. **Select Tags** â­ NEW!
   - Same 6 tags available
   - Select categories that match your project
   - **At least one tag required**

4. **Create Project**
   - Submit the form
   - Project appears with tags

## For Users Browsing Content

### Using Filters on Homepage

1. **View All Content**
   - By default, "All" filter is selected
   - You see all posts and projects

2. **Filter by Category**
   - Click any filter button at the top:
     - **All Posts** - See everything
     - **Technical** - Only tech-related content
     - **Cultural** - Only cultural content
     - **Academic** - Only academic content
     - **Social** - Only social content
     - **Sports** - Only sports content
     - **Other** - Miscellaneous content

3. **What Happens**
   - **Posts section** filters to show only matching posts
   - **Projects section** filters to show only matching projects
   - Both update simultaneously! ğŸ¯

4. **Finding Specific Content**
   - Want to see tech projects? Click "Technical"
   - Looking for cultural events? Click "Cultural"
   - Want to see everything? Click "All"

## Frequently Asked Questions

**Q: Can I select multiple tags?**  
A: Yes! Select as many as relevant to your content.

**Q: What if I don't select any tags?**  
A: You'll see an error. At least one tag is required.

**Q: Can I change tags after posting?**  
A: Not currently. Choose carefully when creating.

**Q: Do tags affect who sees my content?**  
A: Yes! Users filtering by a specific tag will only see content with that tag.

**Q: What if none of the tags fit?**  
A: Use "Other" for miscellaneous content.

**Q: Do I see my own posts in filtered views?**  
A: Yes, if they have matching tags.
