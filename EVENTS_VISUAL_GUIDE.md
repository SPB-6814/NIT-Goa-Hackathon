## Animation Timeline

### Card Hover (300ms)
```
0ms:    Normal state
        - Poster: scale(1)
        - Shadow: none
        - Overlay: opacity(0)

150ms:  Mid-transition
        - Poster: scale(1.05)
        - Shadow: shadow-glow-md
        - Overlay: opacity(0.5)

300ms:  Fully hovered
        - Poster: scale(1.1)
        - Shadow: shadow-glow-lg
        - Overlay: opacity(1)
        - Title visible
```

### Modal Open (500ms)
```
0ms:    Closed
        - Modal: opacity(0), scale(0.95)
        
250ms:  Fade in
        - Modal: opacity(0.5), scale(0.975)
        - Backdrop: blur(0px)
        
500ms:  Fully open
        - Modal: opacity(1), scale(1)
        - Backdrop: blur(8px)
        - Poster: visible and centered
```

### Stacked Posters (on hover)
```
Normal:  All posters at defined z-index
         z-index: 3, 2, 1 (top to bottom)

Hover:   Hovered poster jumps to top
         z-index: 10
         scale: 1.1
         Other posters: unchanged
```
